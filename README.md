# Stone Paper Scissors

An early C++ console game project built while learning programming fundamentals.

The player chooses Stone, Paper, or Scissors and plays against the computer for a selected number of rounds. The game tracks round winners and displays the final result.

## Features

- Choose number of rounds from 1 to 10
- Player vs computer gameplay
- Random computer choice
- Round-by-round winner detection
- Tracks player wins, computer wins, and draws
- Shows final game winner
- Allows replaying the game

## Concepts Practiced

- C++ basics
- Functions
- Enums
- Structs
- Loops and conditions
- Random numbers
- Console input/output
- Basic game logic

- C++
